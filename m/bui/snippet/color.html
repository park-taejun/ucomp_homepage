color

웹 페이지의 컬러는 사용자에게 이정표와 같은 역할을 하므로 일관성 없이 남용된 컬러의 사용은 사용자에게 혼란을 주게 됩니다.
본 프로젝트의 컬러 시스템을 아래와 같이 정의하며, 반드시 약속된 값을 사용합니다.

color system

SCSS usage

argument
color($theme, $variant, $opacity);
example

border-color: color(base, 060);
background-color: color(base, 000, 0.6);
